# Chess Game UI - React Frontend

Modern React/Next.js frontend for the AI Chess Game project.

Part of the [Chess Game CEF](https://github.com/yourusername/chess-game-cef) project.

## Features
- Interactive chess board with click-to-move
- 8 AI opponents with unique personalities
- Real-time AI commentary
- Sound effects and animations
- Settings panel (sound, chat visibility)

## Tech Stack
- React + Next.js
- TypeScript
- Chess.js
- OpenAI API
- Tailwind CSS
